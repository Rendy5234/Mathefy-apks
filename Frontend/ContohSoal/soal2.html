Sederhanakan penjumlahan bentuk aljabar dari 9 + 7 + x + 8y + 2!

9 + 7 + x + 8y + 2  = ( 9x + x ) + 8y + ( 7 + 2 )
= 10x + 8y + 9

Sederhanakan penguranan bentuk aljabar dari 10x2 - 8x - x - 9y - 6!

10x2 - 8x - x - 9y - 6  = 10x2 + ( -8x - x ) - 9y - 6
= 10x2 - 9x - 9y - 6

Sederhanakan perkalian bentuk aljabar dari 2p x 3p2 x 3r!

2p x 3p2 x 3r  = ( 2 x 3 x 3 )( p x p x r )
= ( 18 )( p3r)
= 18p3r

Sederhanakan pembagian bentuk aljabar dari 3p2 : 3p!

3p2 : 3p  = 3p2/3p
= 3.p.p/3.p
= p

Sederhanakan perpangkatan bentuk aljabar dari ( 3p )2!

( 3p )2  = 3p 3p x 3p
= ( 3 x 3 )( p x p )
= ( 9 )( p2 )
= 9p2
