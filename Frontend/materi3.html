Menulis & Menyederhanakan Bentuk Aljabar

Bentuk aljabar dapat dituliskan berdasarkan simbol-simbol operasinya, seperti penjumlahan, pengurangan, perkalian, pembagian, perpangkatan dan lain-lainya. Berikut cara menulis bentuk-bentuk aljabar sesuai symbol operasi masing-masing!

Pemnjumlahan
a.  x + 0 = x
b.  x + x = 2x
c.  x + x + x = 3x
d.  x + x + 2 = 2x + 2

Pengurangan
a.  x - x = 0
b.  x - x - x = -x
c.  x - x - x - x = -2x
d.  x - x - x - x - 2 = -2x - 2

Perkalian ( x bisa diganti * )
a.  1 x a = a
b.  ( -1 ) x a = -a
c.  4 * a = 4a
d.  ( -4 ) * = -4
e.  3 * a * b = 3ab
f.  ( 4 * a ) + ( 5 * b ) = 4a + 5b

Pembagian
a.  1 : a = 1/a
b.  a : 2 = a/2
c.  ( -3 ) : a = -3/a
d.  a : ( -6 ) = -a/6
e.  ( -2 ) : ( -3 ) = -2/-3 = 2/3

Pemnjumlahan ( x bisa diganti * )
a.  p x p = p2
b.  p x p x p = p3
c.  p * p * r = p2 r
d.  p * p * r * r = p2 r2
e.  ( -p ) * p * r = -p2 r

Berdasarkan contoh penulisan bentuk aljabar diatas sesuai symbol operasinya, aljabar dapat juga disederhanakan. Menyederhanakan bentuk aljabar dapat dilakukan dengan cara mengoperasikan suku-suku sejenis berdasarkan lambang operasinya dan kostantanya. Suku-suku sejenis yang dimaksudkan adalah suku-suku yang memiliki variabel sama walaupum koefisienya berbeda. Seperti 2m sejenis dengan 4m. p2 sejenis dengan 9p2
